编辑完host文件没有权限保存怎么办
发表于 2020-03-31| 更新于 2020-03-31
字数总计:97|阅读时长: 1 分钟
| 阅读量:

1. win+R

2. 进入hosts的文件所在目录，点击‘’文件‘’按钮

3. 点击Windows PowerShell ，在点击以管理员身份打开

4. 弹出一个窗口，输入 cmd 并回车，就进入了管理员界面。输入命令行：notepad hosts 并回车

5. 打开了hosts，然后就可以执行复制粘贴等编辑操作了
